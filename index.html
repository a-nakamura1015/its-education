Git の操作で使用するために学習する必要があるコンテンツである。

主にファイル・フォルダ操作のコマンドをおさえる。

Windows と Mac のどちらでも進められる内容にする。

$ cd Document
$ touch sample.txt
$ rm sample.txt

プロジェクトにアサインされるために必要となる Git の知識を身に着ける。

あまり深追いはせずに、Git の概念（ソース管理の意義）や基本的な Git コマンドをおさえる。

$ git add -A
$ git commit -m "first commit"
$ git push origin master
